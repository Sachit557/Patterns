# Patterns in C++

A beginner-friendly collection of **pattern printing programs in C++**.  
This repository contains classic numeric, character, and star-based patterns implemented using basic loops and logic.

Created mainly for **practice, revision, and reference** — useful for beginners learning:
- nested loops
- spacing logic
- pattern construction
- basic C++ I/O

---

## Files Included

### Number Patterns
- **num_pyramid.cpp**  
  Basic number pyramid pattern.

- **digit_pyramid.cpp**  
  Pyramid using digits in increasing order.

- **number_pyramid_2.cpp**  
  Variation of number pyramid with mirrored structure.

- **inverted_triangle_pyramid.cpp**  
  Inverted numeric triangle pattern.

- **Reverse_triangle_pattern.cpp**  
  Reverse number triangle with spacing.

- **number_sq_pattern.cpp**  
  Square pattern using numbers.

- **numbers_sq_pattern_2.cpp**  
  Alternate square number pattern.

---

### Character Patterns
- **Floyd_char_pyramid.cpp**  
  Floyd’s triangle using characters (A, B, C, ...).

- **Floyds_triangle_pattern.cpp**  
  Classic Floyd’s triangle pattern.

---

### Star Patterns
- **star_pyramid.cpp**  
  Star pyramid pattern.

- **pyramid_pattern.cpp**  
  Symmetric pyramid pattern.

- **Hollow_diamond_pattern.cpp**  
  Hollow diamond star pattern.

- **butterfly_pattern.cpp** *(upcoming)*  
  Butterfly/star pattern using symmetric loops.

---

### Executables

  Compiled executable (Windows).
